OCR Extraction

Clean Parsing

Structured Storage

Invoice data is stored in MySQL via SQLAlchemy ORM for easy querying and scalability.

Dashboard View

Manage invoices visually with a simple, organized dashboard built with FastAPI and Jinja2.

Predictive Insights

Smart Reminders

AI-generated reminder emails that adapt tone and urgency based on payment behavior.
